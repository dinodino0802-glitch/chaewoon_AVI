# 🎮 비디오 플레이어 설정 가이드

## 📍 시계 위치 조정 (app.js)

`app.js` 파일의 **CONFIG** 섹션에서 수정하세요.

```javascript
const CONFIG = {
  // ... 다른 설정들 ...
  
  // === 시계 위치 ===
  timePositionTop: '50%',      // 위쪽 위치
  timePositionLeft: '50%',     // 좌측 위치
  timeTextAlign: 'center',     // 텍스트 정렬
};
```

### 위치 예시

#### 중앙 (기본값)
```javascript
timePositionTop: '50%',
timePositionLeft: '50%',
```

#### 상단 중앙
```javascript
timePositionTop: '20%',
timePositionLeft: '50%',
```

#### 하단 중앙
```javascript
timePositionTop: '80%',
timePositionLeft: '50%',
```

#### 좌측 상단
```javascript
timePositionTop: '20%',
timePositionLeft: '20%',
timeTextAlign: 'left',
```

#### 우측 하단
```javascript
timePositionTop: '80%',
timePositionLeft: '80%',
timeTextAlign: 'right',
```

## 📏 시계 크기 조정

```javascript
const CONFIG = {
  timeDisplaySize: 120,   // 시간 텍스트 크기 (px) - 120 → 100으로 줄이면 작아짐
  timeLabelSize: 24,      // 라벨 텍스트 크기 (px)
};
```

예: 더 작게
```javascript
timeDisplaySize: 80,      // 큰 숫자 = 더 크게, 작은 숫자 = 더 작게
```

## 🎨 시계 색상 변경

```javascript
const CONFIG = {
  timeColor: 'white',                         // 텍스트 색상
  timeShadowColor: 'rgba(0, 0, 0, 0.8)',     // 그림자 색상
  timeGlowColor: 'rgba(102, 126, 234, 0.5)', // 글로우 색상 (빛)
};
```

### 색상 예시

#### 빨간색 글로우
```javascript
timeShadowColor: 'rgba(0, 0, 0, 0.8)',
timeGlowColor: 'rgba(255, 0, 0, 0.5)',  // 빨강
```

#### 녹색 글로우
```javascript
timeShadowColor: 'rgba(0, 0, 0, 0.8)',
timeGlowColor: 'rgba(0, 255, 0, 0.5)',  // 초록
```

#### 노란색 글로우
```javascript
timeShadowColor: 'rgba(0, 0, 0, 0.8)',
timeGlowColor: 'rgba(255, 255, 0, 0.5)',  // 노랑
```

#### 그림자 제거 (밝게)
```javascript
timeShadowColor: 'rgba(0, 0, 0, 0)',     // 투명 = 그림자 제거
timeGlowColor: 'rgba(200, 200, 200, 0.3)',
```

## 🐛 문제 해결

### "시계가 안 보여요"
- `timePositionTop`, `timePositionLeft` 값이 이상한지 확인
- 예: `timePositionTop: '150%'` ← 영상 밖에 있음

### "글로우가 안 보여요"
- `timeGlowColor` 값이 투명하지 않은지 확인
- RGB값을 확인 (rgba(R, G, B, 투명도))
